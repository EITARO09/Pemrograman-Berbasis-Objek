# Aplikasi Biodata Mahasiswa

Program Java sederhana menggunakan Swing untuk menampilkan biodata mahasiswa.

## Cara menjalankan

1. Buka terminal di folder `Tugas 12`.
2. Kompilasi:
   ```
   javac BiodataMahasiswa.java
   ```
3. Jalankan:
   ```
   java BiodataMahasiswa
   ```

## Fitur

- Input: NIM, Nama, Program Studi.
- Tombol `Tampilkan` menampilkan data di area output.
- Tombol `Reset` mengosongkan form dan output.
